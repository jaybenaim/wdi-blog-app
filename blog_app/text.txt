What is a client? 
A client is a computer hardware device or software that accesses a service made by a server

What is a server? 
A server is a designated computer that waits from requests from a client and then provides responses accordingly. The server provides a function or service to one or many clients, which initiates requests for such services.

What is an HTTP request? July 28, 2019, 9:30 p.m.
An HTTP request has three parts; the request line, the headers, and the body. The request line says what the client wants to do, what it wants to do it to, and the protocol its speaking. When a client makes a request the server looks for the matching response and sends the file or whatever the client inquired about.

What is an HTTP response? July 28, 2019, 9:31 p.m.
An HTTP response is what the server sends to the client upon the clients request. It can be the resource that the client requested or an error.
